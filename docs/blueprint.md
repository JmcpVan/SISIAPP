# **App Name**: SISIAPP

## Core Features:

- Service Listing: Display available local services with details like name, description, and team validation status.
- Service Registration: Allow service providers to register their services via a form. Collect necessary information for listing.
- Validation Indicator: Visually indicate whether a service has been validated by the SISIAPP team before being published.
- Service Suggestion Tool: Recommend suitable service categories based on the service description, powered by a generative AI tool.

## Style Guidelines:

- Primary color: Google Blue (#4285F4) for headings and primary buttons, conveying trust and reliability, matching the SISIAPP logo.
- Secondary color: Google Red (#EA4335) for calls to action and important alerts.
- Tertiary color: Google Yellow (#FBBC05) for highlights and less critical actions.
- Quaternary color: Google Green (#34A853) to indicate success and availability.
- Font: 'PT Sans', a versatile humanist sans-serif, for both headlines and body text, ensuring readability and a modern feel.
- Use flat design icons to represent different service categories. Color the icons using Google's Red (#EA4335) and Green (#34A853).
- Use a grid-based layout for service listings, ensuring a clean and organized presentation. Cards should display key service information clearly.
- Subtle transitions and animations on button hover states to provide a responsive user experience.
